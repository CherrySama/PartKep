# PartKep - Part Keypoint Detection for Manipulator

基于 GroundingDINO 的部分关键点检测项目，用于机械臂等应用。

## 权重文件下载（不包含在仓库中）
请手动下载 GroundingDINO 模型权重并放入 `models/GroundingDINO/` 文件夹：

- SwinB 版本（推荐）：[groundingdino_swinb_cogcoor.pth](https://github.com/IDEA-Research/GroundingDINO/releases/download/v0.1.0-alpha/groundingdino_swinb_cogcoor.pth)

其他版本权重见官方 release 页面：https://github.com/IDEA-Research/GroundingDINO/releases
